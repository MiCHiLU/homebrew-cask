# homebrew-cask
A tap for the Homebrew that provides by michilu.

# Install

```console
$ brew tap michilu/homebrew-cask git@github.com:michilu/homebrew-cask.git
```

# Development

```console
$ cd $(brew --repository)/Library/Taps/michilu/homebrew-cask
```
